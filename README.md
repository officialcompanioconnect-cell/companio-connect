# Companio Marketing Website

Premium, animated marketing website for Companio – a social connection app focused on structured, time-bound meetups and real relationships.

## Features
✨ Light/dark theme toggle with smooth transitions
✨ Cinematic hero with animated background effects
✨ Smooth scroll-triggered animations (respects reduced-motion)
✨ Interactive FAQ accordion
✨ Fully responsive (mobile, tablet, desktop)
✨ WCAG AA accessible
✨ Vanilla JS – no frameworks or dependencies

## Tech Stack
- HTML5 (semantic markup)
- CSS3 (Grid, custom properties, animations)
- Vanilla JavaScript (Intersection Observer, localStorage)
- Inter font family

## File Structure
```
project-root/
├── index.html (complete single-page site)
├── assets/
│   ├── css/style.css (theme system + animations)
│   ├── js/script.js (toggle, animations, accordion)
│   ├── images/ (logo, hero, stock photos)
│   └── fonts/
├── README.md
```
